hexo 常用命令

hexo clean

清除缓存文件 db.json 和已生成的静态文件 public。

hexo generate

生成网站静态文件到默认设置的 public 文件夹。
可以简写为 hexo g

hexo deploy

自动生成网站静态文件，并部署到设定的仓库。
可以简写为 hexo d

hexo server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。
可以简写为 hexo s

hexo new “title”

新建一篇文章，可以简写为 hexo n <title>

hexo n page pageName

新建一个标题为 aboutme 的页面，默认链接地址为 主页地址/aboutme/
